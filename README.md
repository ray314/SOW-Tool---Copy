# Scope-of-Works
My R&amp;D project - 
a web tool for quantity surverying that records scope of works for construction projects.

<b>List of technologies used</b>
* Angular
* TypeScript
* Javascript
* Bootstrap 5
* Git
* NodeJS
* Express.js
* SQL Server
